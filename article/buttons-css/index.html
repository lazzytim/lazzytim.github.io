Custom CSS Buttons

I've been testing button designs for quite a while and i fell in love with these little buttons. I hope you'll like them as well!

Import the style sheet

Wanna start using my buttons style sheet?
Nothing is more simple then that. Just add this line in the <head> of your .html file.

<link src="https://lazzytim.github.io/assets/css/buttons.css"></link>

Create the buttons

To create a buttons, use one of the class below:
And replace the $class in the code below.

Name Class ($class)
Basic "basic"
Link "link"
Warning "warning"
Danger "danger"
Destructive "destructive" 
<button class="$class">Exemple Text<button>
